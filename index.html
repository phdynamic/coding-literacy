◆ Introduction
You already
think in code.

You know variables, functions, loops, and logic from your BASIC and C++ days. That mental model is the hard part — and you have it. This course fills the modern gap: the languages and patterns Claude Code actually uses, so you can read what it writes, catch what it gets wrong, and prompt it with precision.

What you're bringing

A working mental model of how programs run. Variables hold data. Functions encapsulate logic. Libraries give you pre-built tools. Control flow routes execution. All of that transfers directly.

What this fills in

Modern syntax and idioms. Web-specific patterns. Async programming. Component-based UI. Config file formats. Shell scripting for deployment. The stuff that looks unfamiliar but maps to things you already know.

How each lesson works

Every lesson starts with the big picture — what this thing does — before getting into how. Annotated examples, reference tables, a Prompt Lab with real AI feedback, and a recognition quiz.

The Prompt Lab

Each lesson ends with a realistic scenario. You write the prompt you'd give Claude Code. Claude evaluates it and tells you what's strong, what's vague, and what you could add to get better output.

  • HTML/CSS Structure and styling — the bones and skin of every web tool you build
  • JS JavaScript — interactivity, API calls, the full logic layer
  • PY Python — scripting, server logic, automation
  • SH Bash/Shell — terminal commands, deployment, Bazzite life
  • CFG JSON & YAML — config files, package.json, deploy manifests
  • JSX React & JSX — components, state, the pattern behind your interactive tools
